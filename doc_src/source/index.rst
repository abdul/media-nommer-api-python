=============================================================
media-nommer-api-python: A Python API client for media-nommer
=============================================================

This module simplifies accessing `media-nommer`_'s JSON API. Request formation,
sending, and response parsing is handled for the developer. One should be able
to read through the `media-nommer`_ documentation and expect to be productive
with this API module in short order.

Purpose
-------

Using this API client along with `media-nommer`_ will allow you to do things
such as:

* Submit encoding jobs
* Check the status of existing jobs

Learning more
-------------

Make sure to read through the `media-nommer`_ documentation.
